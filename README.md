# node-api
Node.JS study

<h2>Filters</h2>
Filter by name
```
GET /name="value"
```
Filter by positive market variation
```
GET /up=true
```
Filter by negative market variation
```
GET /down=true
```
Ascending order
```
GET /order=asc
```
Descending order
```
GET /order=des
```
Return all data
```
GET /all=true
```
